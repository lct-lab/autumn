# Autumn
**Autumn** is a cpp framework to operate database with odbc, and providing rpc services with thrift in a code-less way.  Free yourself from the tedious basic CURD work.

**Supported database:**
* postgreSQL

**requirement:**
* thrift
* boost
* inicpp
* log4cplus
* odbc
